# Distributions Linux issues de Red Hat

## 1. Red Hat Enterprise Linux (RHEL)

-   Distribution commerciale entreprise
-   Support long terme (LTS)
-   Forte stabilité et certifications professionnelles
-   Base principale des autres distributions dérivées

## 2. Fedora

-   Distribution communautaire sponsorisée par Red Hat
-   Cycle de publication rapide
-   Sert de laboratoire d'innovations pour RHEL

## 3. CentOS (historique)

-   Clone gratuit et compatible binaire de RHEL
-   Très utilisé en environnement serveur
-   Remplacé par CentOS Stream

## 4. CentOS Stream

-   Version intermédiaire entre Fedora et RHEL
-   Sert de préversion continue de RHEL

## 5. Rocky Linux

-   Distribution communautaire compatible RHEL
-   Créée après la fin du CentOS classique
-   Alternative gratuite orientée stabilité

## 6. AlmaLinux

-   Fork communautaire de RHEL
-   Compatible RHEL
-   Très utilisé en hébergement et serveurs web

## 7. Oracle Linux

-   Basé sur RHEL
-   Maintenu par Oracle
-   Propose son propre noyau (UEK)

------------------------------------------------------------------------

## Schéma simplifié

Fedora → RHEL → (Rocky Linux, AlmaLinux, Oracle Linux) ↑ CentOS Stream

------------------------------------------------------------------------

## Résumé rapide

-   RHEL : version entreprise stable
-   Fedora : innovation et nouvelles technologies
-   CentOS Stream : préversion continue de RHEL
-   Rocky Linux / AlmaLinux : alternatives gratuites compatibles RHEL
-   Oracle Linux : version entreprise modifiée par Oracle
